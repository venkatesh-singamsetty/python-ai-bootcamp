# Day 51: Customer Segmentation Analytics

This project implements customer segmentation using K-Means and Hierarchical clustering with an interactive Streamlit dashboard.

## 🚀 Quick Start Guide (Using Root VENV)

```bash
# Ensure you are in the project folder and root .venv is active
python kmeans_clustering_model.py
python hierarchical_clustering_model.py
python -m streamlit run streamlit_customer_segmentation_dashboard.py
```

## 📁 Project Overview
*   **k-Means Clustering**: `kmeans_clustering_model.py` (Elbow Method & Centroids)
*   **Hierarchical Clustering**: `hierarchical_clustering_model.py` (Dendrogram & Agglomerative)
*   **Interactive App**: `streamlit_customer_segmentation_dashboard.py` (Real-time exploration)
*   **Dataset**: `Mall_Customers.csv`
