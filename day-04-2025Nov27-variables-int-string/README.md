## 🔢 Python Fundamentals

### 1. Mathematical Operations (BODMAS)
All numerical calculations in our models follow the **BODMAS** rule to ensure accuracy:
* **B**rackets
* **O**rders (Exponents)
* **D**ivision / **M**ultiplication
* **A**ddition / **S**ubtraction

### 2. Data Types (Non-Technical to Programming)
| Concept | Data Type | Usage |
| :--- | :--- | :--- |
| **Numbers** | `int` / `float` | Used for ages, prices, and coordinates. |
| **Text** | `str` (String) | Used for names, reviews, and labels. |
| **Previous Output** | `_` | In Python, the underscore stores the result of the last execution. |

### 3. Working with Text (Strings)
* **Single/Double Quotes (`' '` or `" "`):** Used for single-line text data.
* **Triple Quotes (`''' '''`):** Used for multi-line strings.
    * *Practical Use Case:* Cleaning large text files (e.g., movie scripts/datasets) or writing multi-line comments for AI program logic.
